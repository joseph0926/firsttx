# 문서 인덱스

이 폴더는 저장소 문서를 보관합니다. AI 친화적 진입점은 `docs/ai/`에 있습니다.

## 시작하기

- `docs/ai/README.md`: AI 온보딩, 읽기 순서, 빠른 명령
- `docs/ai/repo-map.md`: 워크스페이스 맵 + 핵심 파일
- `docs/ai/architecture.md`: 제품 아키텍처 개요
- `docs/ai/apps.md`: 앱 레벨 노트 (문서 사이트, 플레이그라운드)
- `docs/ai/packages.md`: 패키지 레벨 노트
- `docs/ai/workflows.md`: 빌드/테스트/릴리스 워크플로
- `docs/project-analysis.md`: 상세 분석 (한국어)

## 관련 문서

- `README.md`: 프로젝트 개요
- `apps/docs/README.md`: 문서 앱 안내
- `apps/playground/README.md`: 플레이그라운드 개요
- `packages/*/README.md`: 패키지별 문서
